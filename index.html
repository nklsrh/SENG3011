for Financial Trading

    • Windows 7/8 64-bit

    with Java 6 (64-bit) and Java 7 (32-bit/64-bit)

  • What's new in this release?

    Implementation

    It selects trade transactions from a list of transactions in the CSV file, then calculates its return over time.
    The functions are logged and saved to a file called MomentumStrategyModule.log, a log report containing details about the file (number of TRADES, LINES, execution time), which is saved to the working directory.

    Updates

    This is the first version.

    Additional files

    • inputFile.csv, the test input file with Sirca data
    • arguments.txt, the parameters text file (currently empty)

    Note: As of v1.0, the module does not yet use params specified in arguments.txt, but the file is required to exist.

To run
  • 1. Open cmd and go to the downloaded JAR's directory
  • 2. Type the following:

    java -jar MSM.jar inputFile.csv arguments.txt

    Where
    java -jar is the function to run a JAR file
    MSM.jar is the program module to be run
    inputFile.csv is the input file
    arguments.txt is a text file which includes parameters needed for the module

  • 3. The functions are logged and saved to a file called MomentumStrategyModule.log in the same directory.
Note: The arguments.txt file can be left blank.
Note: As of v1.0, the module does not yet use params specified in arguments.txt, but the file is required to exist
